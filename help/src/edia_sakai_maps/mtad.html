Maps: Tool Administration

The administration page is used for entering the Google Maps keys and for configuring the three 'naviagation maps' located on right the side of the tool. Administrators can enter the following information:

  • For each of the three 'naviagation maps', the following properties can be set:
    • The name of the map
    • The latitude of the map's center
    • The longitude of the map's center
    • The zoom level of the map
    • The map type of the map
    • The image location of the actual image on the right side, this can any valid url.
  • The default map: the initial map to display.
  • Optionally, a set of Google Maps API keys. The Sakai Maps tool will not work without a Google Maps key.
    Keys can be configured server-wide in the sakai.properties file or individually for every sakai maps tool placement. Values set specifically for a tool override the default values set in the sakai.properties file.
    A Google Maps key, which you can get from Google, is bound to a url. If your map is reachable by several URLs, you will need a key for each url. Please note that the form does not validate the key, an invalid key is accepted and will result in an error message when opening the maps page.
    If there aren't any keys defined, only an Add button will be displayed. Click that to add the first key. The URL field is optional if you're server is only accessible under a single url. You can add more keys by clicking Add again and you can remove specific keys by clicking the Delete button.

N.B. You can obtain these map configuration values using Google Maps on the web. First, navigate to your desired map. Then click on the 'link to this page' link just above the right top corner of the map. The link will appear in your browsers address bar. The next example illustrates how the values appear in the generated link:

http://maps.google.com/maps?ie=UTF8&om=1&ll=52.389849,4.923248&spn=0.191505,0.462799&z=11

Latitude and longitude appear in the ll parameter of the URL as ll=lat,long. In the example above these values are: lat=52.389849 and long=4.923248

Zoom level appears in the z parameter as z=zoom. In the examples above: zoom=11

The image must be linked using a relative path to a location inside Sakai (e.g. from the resource tree) or using a full path to a location outside Sakai (must start with http://).
